# OpenLeave - Leave Management System 🌴

Το **OpenLeave** είναι ένα ελαφρύ, Open-Source σύστημα διαχείρισης αδειών προσωπικού, σχεδιασμένο για μικρομεσαίες επιχειρήσεις. 

## Το Πρόβλημα που λύνει
Αντικαθιστά τα χαρτιά και τα Excel, προσφέροντας ένα απλό ψηφιακό περιβάλλον όπου οι υπάλληλοι ζητούν άδεια και οι managers την εγκρίνουν.

## Τεχνολογίες
- HTML5
- CSS3
- JavaScript (Vanilla)

## Εγκατάσταση & Χρήση
Απλώς κατεβάστε τον κώδικα, ανοίξτε το αρχείο `index.html` στον browser σας (Chrome/Firefox/Edge) και είστε έτοιμοι! Δεν απαιτείται εγκατάσταση server.
