# Contributing to OpenLeave

Σας ευχαριστούμε που ενδιαφέρεστε να συνεισφέρετε στο OpenLeave! Ως open-source project, καλωσορίζουμε τη βοήθεια από την κοινότητα.

## Πώς να συνεισφέρετε
1. Κάντε **Fork** το repository.
2. Δημιουργήστε ένα νέο branch για το χαρακτηριστικό σας (`git checkout -b feature/AmazingFeature`).
3. Κάντε Commit τις αλλαγές σας (`git commit -m 'Add some AmazingFeature'`).
4. Κάντε Push στο branch (`git push origin feature/AmazingFeature`).
5. Ανοίξτε ένα **Pull Request** και η ομάδα μας θα το ελέγξει.

## Κανόνες Κώδικα (Code of Conduct)
- Παρακαλούμε να γράφετε καθαρό κώδικα με σχόλια.
- Σεβαστείτε τα υπόλοιπα μέλη της κοινότητας κατά την επικοινωνία στα Issues.
- Όλα τα Pull Requests πρέπει να εγκρίνονται από τουλάχιστον 2 μέλη της βασικής ομάδας πριν γίνουν merge.
