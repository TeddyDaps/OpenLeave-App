# Security Policy

## Supported Versions
Αυτή τη στιγμή, οι παρακάτω εκδόσεις του OpenLeave λαμβάνουν ενημερώσεις ασφαλείας:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability
Η ασφάλεια του συστήματος διαχείρισης αδειών είναι ύψιστης σημασίας. 
Αν ανακαλύψετε κάποιο κενό ασφαλείας (vulnerability) στο OpenLeave, **παρακαλούμε ΜΗΝ ανοίξετε δημόσιο Issue**. 

Αντίθετα, στείλτε ένα email απευθείας στην ομάδα διαχείρισης στο: `security@openleave-example.com` (ή στο email του Project Manager). 
Θα επιβεβαιώσουμε τη λήψη του μηνύματός σας εντός 48 ωρών και θα σας ενημερώνουμε για την πορεία επίλυσης του προβλήματος πριν βγάλουμε το επίσημο patch.
