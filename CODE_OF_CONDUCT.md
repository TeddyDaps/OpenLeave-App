# Contributor Covenant Code of Conduct

## Η Δέσμευσή μας
Στοχεύοντας στη δημιουργία ενός ανοιχτού και φιλόξενου περιβάλλοντος, εμείς ως συνεισφέροντες (contributors) και διαχειριστές (maintainers) του **OpenLeave** δεσμευόμαστε να κάνουμε τη συμμετοχή στο project μας μια εμπειρία χωρίς παρενοχλήσεις για όλους, ανεξαρτήτως ηλικίας, φύλου, σεξουαλικού προσανατολισμού, εθνικότητας ή τεχνικού επιπέδου.

## Τα Πρότυπά μας
Παραδείγματα συμπεριφοράς που συνεισφέρουν στη δημιουργία ενός θετικού περιβάλλοντος περιλαμβάνουν:
* Χρήση φιλικής και χωρίς αποκλεισμούς γλώσσας.
* Σεβασμός στις διαφορετικές απόψεις και εμπειρίες.
* Εποικοδομητική αποδοχή της κριτικής (code reviews).

Παραδείγματα μη αποδεκτής συμπεριφοράς:
* Χρήση σεξουαλικοποιημένης γλώσσας ή εικόνων.
* Προσβλητικά σχόλια (trolling) και προσωπικές ή πολιτικές επιθέσεις.
* Δημόσια ή ιδιωτική παρενόχληση.

## Επιβολή (Enforcement)
Περιστατικά καταχρηστικής ή παρενοχλητικής συμπεριφοράς μπορούν να αναφέρονται επικοινωνώντας με τον Project Manager. Η ομάδα διαχείρισης έχει το δικαίωμα και την ευθύνη να αφαιρέσει, να επεξεργαστεί ή να απορρίψει σχόλια, commits και issues που δεν ευθυγραμμίζονται με αυτόν τον Κώδικα Συμπεριφοράς.
